3. Verify the header says "ASK JACK'S AI" with a close button

**Gate 2: Chat persistence test**
1. Open chat and send a message (e.g., "Hello")
2. Wait for bot response to complete
3. Navigate to a different page (e.g., /about or /projects)
4. Open the chat panel again
5. Verify previous messages (your message + bot response) are replayed
6. Verify the privacy note says "Conversations stored locally for 24h."
7. (Optional) Open browser devtools > Application > Local Storage -- verify "chat-history" key exists with messages array

**Gate 3: Visual parity check**
1. Open homepage at 1440px width -- compare hero, work list, about preview, contact section against mockup.html
2. Open homepage at 375px width -- verify responsive layout (hero stacks vertically, sections compress)
