# Phase 6 — UI Design Contract

> Visual and interaction contract for Phase 6: Performance Audit & Deployment. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Phase Nature: Non-Visual

Phase 6 introduces **zero new UI components, zero new pages, and zero new visual elements**. It is a performance optimization, accessibility audit, and deployment verification phase. The design contract below documents the **existing design system as the compliance baseline** that all optimizations must preserve. No visual regressions are permitted.

The only visual change in this phase is reducing mobile canvas particle count (400 to ~200), which is a performance optimization within the existing CanvasHero component. This does not alter the design language.

## Copywriting Contract

Phase 6 adds no user-facing copy. All existing copy is preserved without modification. The contract below documents the **zero-new-copy constraint** and the only edge case.

| Element | Copy |
|---------|------|
| Primary CTA | not applicable -- no new CTAs in this phase |
| Empty state heading | not applicable -- no new empty states |
| Empty state body | not applicable |
| Error state | not applicable -- no new user-facing error states |
| Destructive confirmation | not applicable -- no destructive actions |

### Pre-animation visibility fallback (new in Phase 6)

If GSAP lazy-loading fails (network error, Safari import bug), animated elements must become visible after 3 seconds via JS timeout fallback. No user-facing error message is shown. Content simply appears in its final state without animation. This is a graceful degradation pattern, not an error state.

### prefers-reduced-motion behavior (enforced in Phase 6)

When `prefers-reduced-motion: reduce` is active, all `[data-animate]` elements display at `opacity: 1` immediately via CSS. GSAP is never imported. Canvas particles are fully disabled. No "animations disabled" message is shown to the user -- content appears statically as if animations were never configured. This is per D-05.
