    Create `src/content.config.ts` at the project src/ root (NOT inside src/content/ -- Astro 6 requires this location per Research Pitfall 2):

    ```typescript
    import { defineCollection } from "astro:content";
    import { glob } from "astro/loaders";
    import { z } from "astro/zod";

    const projects = defineCollection({
      loader: glob({ pattern: "**/*.mdx", base: "./src/content/projects" }),
      schema: ({ image }) =>
        z.object({
          title: z.string(),
          tagline: z.string().max(80),
          description: z.string(),
          techStack: z.array(z.string()).min(1),
          featured: z.boolean().default(false),
          status: z.enum(["completed", "in-progress"]),
          githubUrl: z.string().url().optional(),
          demoUrl: z.string().url().optional(),
          thumbnail: image(),
          category: z.enum(["web-app", "cli-tool", "library", "api", "other"]),
          order: z.number().int().min(1),
        }),
    });

    export const collections = { projects };
    ```

    Schema field rationale (per D-07 through D-10):
    - `title`: z.string() -- project name, required
    - `tagline`: z.string().max(80) -- ~10 words for card scanning (D-08), capped at 80 chars
    - `description`: z.string() -- longer intro for case study pages (D-08)
    - `techStack`: z.array(z.string()).min(1) -- at least one technology, array of strings
    - `featured`: z.boolean().default(false) -- for homepage featured section, defaults to not featured
    - `status`: z.enum(["completed", "in-progress"]) -- two valid states
    - `githubUrl`: z.string().url().optional() -- optional GitHub link, validated as URL if provided
    - `demoUrl`: z.string().url().optional() -- optional demo link, validated as URL if provided
    - `thumbnail`: image() -- Astro image optimization helper, validates image path exists
    - `category`: z.enum(["web-app", "cli-tool", "library", "api", "other"]) -- fixed categories
    - `order`: z.number().int().min(1) -- manual sort control (D-09), positive integer

    CRITICAL: Import `z` from `astro/zod` (NOT from `zod` -- see Research Pitfall 3). Import `glob` from `astro/loaders`. Import `defineCollection` from `astro:content`. Use `image()` helper from schema function parameter for Astro-optimized image validation.

    Per D-10: No role, team size, or duration fields.
