    **HTML Structure:**

    Key structural decisions:
    - Canvas is `aria-hidden="true"` (decorative)
    - Content overlays canvas with `relative z-10`
    - Uses `<slot />` so the parent page (index.astro) passes in the name/role/intro text
    - Full viewport height (`h-screen`)
    - The `justify-end` + padding positions content toward the bottom (inspect shiyunlu.com to confirm where text sits relative to canvas -- adjust if text is centered instead)

    **Script block (generative art engine):**

    The script tag imports simplex-noise and implements a flow field particle system:

    ```javascript
    import { createNoise2D } from 'simplex-noise';
    ```

    **Canvas initialization:**
    1. Get canvas element by id `hero-canvas`
    2. Get 2D rendering context
    3. Account for devicePixelRatio (DPR):
       ```javascript
       const dpr = window.devicePixelRatio || 1;
       canvas.width = window.innerWidth * dpr;
       canvas.height = window.innerHeight * dpr;
       canvas.style.width = window.innerWidth + 'px';
       canvas.style.height = window.innerHeight + 'px';
       ctx.scale(dpr, dpr);
       ```
    4. Create noise function: `const noise2D = createNoise2D();`

    **Particle system:**
    - Create an array of particles (800-1200 on desktop, 300-500 on mobile where `window.innerWidth < 768`)
    - Each particle has: x, y, prevX, prevY properties
    - Initialize particles at random positions

    **Animation loop (draw function):**
    1. Draw a semi-transparent background fill each frame (creates trail effect):
       `ctx.fillStyle = 'rgba(R, G, B, 0.03)';` where RGB matches the `--token-bg-primary` color
       `ctx.fillRect(0, 0, width, height);`
    2. For each particle:
       - Calculate flow angle from noise: `noise2D(x * noiseScale, y * noiseScale + time * 0.0001) * Math.PI * 2`
       - Use `noiseScale = 0.003` (adjust for visual density)
       - Move particle along angle with speed ~1.5
       - Draw line from prevPosition to currentPosition
       - Stroke color should be the accent color at low opacity (e.g., `rgba(accent, 0.15)`) -- extract the accent color from computed styles or hardcode the extracted oklch equivalent
       - Wrap particles around screen edges
    3. Increment time and call `requestAnimationFrame(draw)`

    **Resize handler:**
    - On window resize, recalculate canvas dimensions with DPR
    - Reinitialize particle positions

    **prefers-reduced-motion:**
    ```javascript
    const prefersReducedMotion = window.matchMedia('(prefers-reduced-motion: reduce)').matches;
    if (prefersReducedMotion) {
      // Draw a single static frame: render all particles once without animation loop
      // Just draw scattered dots or a single noise-based gradient
    } else {
      requestAnimationFrame(draw);
    }
    ```

    **View Transitions cleanup (CRITICAL):**
    ```javascript
    let cleanup: (() => void) | undefined;

    document.addEventListener('astro:page-load', () => {
      cleanup?.();
      cleanup = initCanvas() || undefined;
    });

    document.addEventListener('astro:before-preparation', () => {
      cleanup?.();
      cleanup = undefined;
    });
    ```

    The `initCanvas()` function returns a cleanup function that:
    - Calls `cancelAnimationFrame(animationId)`
    - Removes the resize event listener
