## Problem

This case study is being written. Check back soon for the full story. Node.js projects accumulate dependencies quickly, and it is hard to know which packages introduce risk. I wanted a tool that could analyze a project's dependency tree and surface both structural issues and known vulnerabilities.

## Solution & Approach

This case study is being written. Check back soon for the full story. PkgLens reads a project's package.json and lock file, builds the full dependency graph, identifies circular dependencies, and cross-references packages against vulnerability databases.

## Tech Stack Detail

This case study is being written. Check back soon for the full story. The tool is written in TypeScript and runs as a Node.js CLI. It queries the npm Registry API for package metadata and uses a graph data structure internally to detect cycles and calculate dependency depth.
